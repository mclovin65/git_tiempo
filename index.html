1 Git clone Crear una copia del repositorio
2 Git status Muestra el estado del repositorio
3 Git add Añade cambios al área de preparación

6 Git pull Obtiene cambios del repositorio remoto y los fusiona con el repositorio local
7 Git branch Muestra las ramas en el repositorio
8 Git checkout Cambia de rama o restaura archivos
9 Git merge Fusiona una rama con otra
10 Git log Muestra el historial de commits
